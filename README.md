## ⚙️ Configuration

Once installed, locate the nitro-imessage-agent icon on your desktop or in your start menu. Open the program to access the setup screen.

### Setting your preferences

The application requires your authorization to interact with your messaging platform. You will see a login box when you first open the app. Enter your credentials to link your account. The application stores these securely on your device. 

The settings menu allows you to adjust how the agent responds. You can choose:

* Response speed: Determine how fast the agent writes back.
* Active hours: Set the times when the agent should answer messages.
* Language style: Pick the tone for your assistant, such as professional or casual.

Select the "Save Settings" button after making any changes. The agent updates these rules immediately.

## 🚀 Operating the Agent

The main dashboard shows the current status of your agent. A green indicator means the system is active and ready to receive messages. A red indicator means the system is paused.

### Managing message flow

The system keeps a log of all interactions. You can view this log by clicking the "History" tab. This page displays the messages received and the replies sent by the AI. Review these logs if you notice the agent sending unexpected content. You can clear the history at any time using the "Delete Logs" button.

### Refinement of replies

If the agent sends replies you dislike, change the instructions in the "Agent Behavior" tab. Here, you define specific phrases the AI should use or avoid. Providing clear examples helps the agent learn your preferences. For instance, you can tell the agent to "keep all replies under two sentences" or "always use a polite tone."

## 📈 Monitoring Performance

The application includes tools to help you track how your agent performs. Use the "Observability" tab to check how many messages the agent processed today. This section shows:

* Total messages sent.
* Average response time.
* Error counts.

Use this data to decide if you need to adjust your settings. If the error count is high, check your internet connection or verify your login credentials.

## 🔒 Privacy and Security

Your privacy remains a priority. The nitro-imessage-agent processes your data on your machine whenever possible. We do not sell your personal information or read your private messages. ## 📝 Common Questions

Will this application slow down my computer? 
The agent uses a small amount of system power while running in the background. Most modern computers handle this background task without any noticeable effect on speed.

Can I run multiple agents at once?
The current version supports one agent profile per installation. To use multiple agents, you would need to set up separate user accounts on your Windows machine.

Does the agent work if I turn off my computer?
The application must remain open and your computer must be turned on for the agent to reply to messages. If you close the app or sleep your computer, the agent pauses its work. Once you turn the machine back on, the agent resumes processing any pending messages that arrived while you were away.
